## Getting started

Make sure you have the following installed globally: NodeJS, Git, TypeScript, Angular CLI.

## Development server

```bash
$ cd server
$ npm i -g gulp-cli
$ npm i
$ npm build
$ npm start
```

## Angular Client

```bash
$ cd client
$ npm i
$ ng serve
```
Navigate to [http://localhost:4200](http://localhost:4200/).
